# BeatsByMax

Musical improv freestyle practice tool. Static, single-page, no build step.

## Features

- Transport: play/stop, master volume, tap tempo, BPM 60–240, swing 0–66%
- 5 drum presets (Boom Bap, Trap, Lo-fi, Drill, Jazz-hop) — drumpreset only sets pattern/BPM/swing
- 16-step sequencer for kick, snare, hi-hat, open hat with per-track volume + mute (mobile: behind a mixer popover)
- Chords: key, scale (major, minor, dorian, phrygian, mixolydian, pent-minor), 1/2/4 chords per bar
- 5 rap progression presets, randomize, mute-all, click any slot to change degree

## Run

Open `index.html`. Audio starts on first Play.

## Files

- `index.html`
- `styles.css`
- `app.js`

All audio synthesized via Web Audio API.
